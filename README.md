# blogs

Hand-written, single-file interactive blog pages. No framework, no build step — plain HTML, CSS and vanilla JS, deployed with GitHub Pages.

## Design

Every page uses the shared **Motorsport Livery** design system in [`DESIGN.md`](DESIGN.md): carbon black foundation, a single racing-red accent reserved for interaction, Oswald + Inter typography, flat surfaces, near-square corners.
